Rubric for Flask-Intro

topics

Flask

further study concepts

creating virtual envs, using flask routes

requires

import, flask, python, command line, request

time

1 hour

authors

Goals

Students will practice modifying flask routes and interacting with the HTML

Gotchas

Code Reviews

Code Review 1

  • Students have:

    • Created a link to the /hello route on their Flask server

    • Modified /hello to allow a user to select a compliment from a dropdown or radio button

    • Looked at the Flask debugging tools (what can you do when you get into a Flask traceback?) and compared error messages when debug=False

  • Review what the virtualenv does and why students would want a .gitignore file (and what pip freeze > requirements.txt does)

  • Some students have gotten confused about when they need to pip install vs. when they need to source their environments. Specifically, some students in Grace Summer ‘16 thought they needed to pip install every time they sourced their virtualenvs, so might reinforce here that they only need to pip install once and then the next times they source the environment, everything is already installed.

  • Discuss the Flask traceback. What are the specific tools that it provides? (Ability to drop into any point in the code)

Code Review 2

  • Students created a new route called /diss that dishes out a random selection from a list of insults. Pattern matching.

Further Study

  • Using Python, generate a string of valid HTML that includes all inputs from the list of compliments or insults.

  • Add other input types and integrate that user input into the server response

  • Mainly, students are asked to appreciate how annoying it is to serve up HTML directly and look forward to when they’ll learn about rendering templates.

Even Further Study

  • Students created more branches in their app to let users choose how thouroughly complimented or insulted they’d like to be before they pick a compliment.

If Students Finish Even Further Study

If students get past the even further study with time on their hands, here are some suggestions to keep them working in Flask:

  • Recreate calculator, guessing game, or Markov Twitter in Flask.

  • Build an adventure game like you did for the prework, but with Flask.
